# Hidden Layer — Estrutura de Séries (Oficial)

A marca organiza estampas em **séries conceituais**.
Cada série explora uma dimensão da ideia de transformação invisível.

As séries não são coleções temporais — são pilares permanentes.

---

# 1. Mathematics Series

## Tema

Transformação contínua, abstração e progressão.

## Estampas

* GRADIENT
* LIMIT
* SEQUENCE
* INTEGRAL

## Linguagem visual dominante

Categoria A — Bars
Categoria B — Grid

## Ideia central

A matemática descreve processos invisíveis.
As estampas representam estados intermediários.

## Direção de design

* Progressão visual
* Variação incremental
* Ritmo
* Continuidade

---

# 2. System Series

## Tema

Arquitetura interna de sistemas.

## Estampas

* INPUT → HIDDEN → OUTPUT
* LATENT STATE

## Linguagem visual dominante

Categoria B — Grid
Categoria C — Path

## Ideia central

O que acontece dentro de um sistema não é visível, mas define o resultado.

## Direção de design

* Blocos
* Camadas
* Conexões estruturais
* Interrupções

---

# 3. Signal Series

## Tema

Movimento da informação.

## Estampas

* SIGNAL PATH

## Linguagem visual dominante

Categoria C — Path

## Ideia central

Informação percorre trajetórias invisíveis.

## Direção de design

* Fluxo
* Direção
* Conexões
* Continuidade

---

# Regras globais das séries

## Consistência

Toda nova estampa deve pertencer a:

* uma série existente
  ou
* uma nova série claramente definida

Nunca peças isoladas.

---

## Signature vs Statement dentro das séries

Cada série deve ter:

* versões Signature (peito)
* ocasionalmente uma Statement

Exemplo:
GRADIENT → Signature + Statement
LATENT STATE → Signature + futura Statement

---

## Nomeação

Formato padrão:

Série: Nome

Ex:
Mathematics: GRADIENT
System: LATENT STATE
Signal: SIGNAL PATH

---

# Expansão futura natural

Possíveis séries coerentes:

* Noise Series → erro e variabilidade
* Optimization Series → ajuste e convergência
* Representation Series → embeddings e espaço latente
* Temporal Series → sequência no tempo

Criar apenas quando houver 2–3 estampas.

---

# Regra mestre

Séries representam dimensões do mesmo fenômeno:

**transformação interna invisível**

Se uma ideia não conversa com isso → não entra.

---

# Filosofia

Hidden Layer não lança camisetas.
Ela documenta estados de sistemas.
