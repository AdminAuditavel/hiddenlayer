# Hidden Layer — Sistema Visual Oficial

## Princípio central

Hidden Layer comunica o que acontece **entre entrada e saída** — transformação invisível, estrutura interna e fluxo de informação.

A marca não usa ilustração literal.
Ela usa **diagramas abstratos**.

---

# Arquitetura de produto

A marca opera com dois níveis:

## 1. Signature Pieces (80%)

Peças discretas que formam a assinatura da marca.

### Especificação

* Área: **8×8 cm**
* Posição: peito esquerdo
* Elementos: 3–6
* Espessura mínima de linha: **0,7 mm**
* Alto contraste

Objetivo: consistência e identidade.

---

## 2. Statement Pieces (20%)

Peças de impacto visual.

### Especificação

* Área: 18–28 cm largura
* Frente maior ou composição central
* Mesma linguagem das Signature
* Nunca virar ilustração

Regra:
**Statement = Signature ampliada**

---

# Linguagem visual base

Toda estampa precisa pertencer a uma das quatro categorias.

---

## A — Bars (progressão)

Representa estados e transformação.

Exemplos:

* Gradient
* Training
* Activation

Elementos: barras, níveis, variação.

---

## B — Grid (estrutura invisível)

Representa organização interna.

Exemplos:

* Latent State
* Hidden Map

Elementos: grades, quadrados, interrupções.

---

## C — Path (fluxo)

Representa movimento da informação.

Exemplos:

* Signal Path
* Routing

Elementos: linhas, conexões, nós.

---

## D — Noise (variação)

Representa erro, ruído e imprevisibilidade.

Exemplos:

* Noise
* Perturbation

Elementos: pontos, dispersão, irregularidade.

---

# Regras de composição

* Poucos elementos
* Espaçamento respirado
* Geometria simples
* Sem microdetalhe
* Evitar simetria perfeita sempre
* Parecer um diagrama técnico

Pergunta de validação:

**“Isso parece um sistema invisível?”**

Se sim → pertence à marca.

---

# Tipografia padrão

* Grotesk neutra
* Caixa alta
* Tracking levemente aberto
* Palavra curta (1–2)

Exemplos:

* GRADIENT
* LATENT
* SIGNAL
* NOISE

Tipografia é legenda, não protagonista.

---

# Paleta

Base:

* Preto
* Branco
* Cinza técnico

Acento (uso raro):

* Verde neural
* Azul elétrico
* Off-white

Regra: restrição gera identidade.

---

# Estrutura de coleção

Cada coleção deve conter:

* 3–5 Signature
* 1 Statement

Nunca fugir disso.

---

# Filosofia do sistema

Hidden Layer não cria estampas isoladas.
Ela constrói uma linguagem repetível.

Objetivos:

* Consistência
* Atemporalidade
* Sofisticação discreta
* Escalabilidade

---

# Regra mestre

Toda peça deve parecer:

**um fragmento de um sistema maior.**

Se a peça funciona sozinha mas sugere continuidade → está correta.
